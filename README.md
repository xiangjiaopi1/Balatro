# Balatro (simplified)

这是一个极简的 Balatro 灵感项目，只包含标准扑克牌的抽牌与计分。没有小丑、星球或塔罗，只需挑选手牌并计算得分。

## 安装与运行

1. 确保已安装 Python 3.10+。
2. 在仓库根目录运行：

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e .
```

3. 启动命令行演示：

```bash
balatro
```

程序会提供 8 张手牌，输入 5 个索引即可计算出牌型、筹码和倍率。

或启动简易图形界面：

```bash
balatro-ui
```

界面会展示 8 张手牌，点击 5 张后点击“打出所选 5 张”即可查看计分。

## 核心规则

- 牌型判定基于标准 5 张扑克牌规则：同花顺、四条、葫芦、同花、顺子、三条、两对、一对、高牌。
- 每个牌型对应固定筹码与倍率，最终得分 = 筹码 × 倍率。
- 打出 5 张牌后会尝试补回至 8 张手牌，直至牌堆耗尽。
